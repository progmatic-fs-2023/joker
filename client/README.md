# vite-react-frontend

### használata:

- **telepítés:**
    ```shell
    $ npm install
    ```
- **fejlesztői verzió elindítása:**
    ```shell
    $ npm run dev
    ```
### backend api url megváltoztatása:
- hozz létre egy `.env` file-t a project gyökerében a `.env.example` mintájára.
- A `VITE_BACKEND_URL` kulcsnak add meg a saját backend-ed URL-ét.
